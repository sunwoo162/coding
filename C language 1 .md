# 여기는 언어 공부
## 자료형
```
문자형 : char 1byte :-128 ~ 127
부호 없는 문자형 : unsigned char 1 : 0 ~ 255
정수형 : int 2 : 2-15~2 15-1
정수형 : unsigned int 2,4 : 0 ~ 2 16-1,0 ~ 2 32-1
short 정수형 : short 2 : 2-15 ~ 2 15-1
long 정수형 : long 4 : 2-31~2 31-1
long 정수형 : unsigned long 4 : 0 ~ 2 32-1
실수형(단정도 부동 소수점형) : float : 4byte : +-3.4 X 10-38 ~ +-3.4 X 10 38
실수형(배정도 부동 소수점형) : double : 8
실수형(배정도 부동 소수점형) : long double : 16
무형 : 무지형 : void : 값이 없음
```
## 형식 지정 문자
```
%o : 8진수 출력
%d : 10진수 출력
%ld : long형의 10진수 출력
%x, %X : 16진수 출력(%x는 소문자,%X는 대문자로 출력)
%f : 소수점이 있는 실수 출력(float)
%e, %E : 지수 형식으로 출력(%e는 소문자 %E는 대문자로 출력)
%c : 한 글자 문자 출력(char)
%s : 문자열 출력(string)
%u : 부호 없는 10진수 출력(unsigned int, unsigned short)
%lu : 부호 없는 long형의 10진수 출력(unsigned long)
%lf : 소수점이 있는 실수 출력(double)
```
## 연산자
```
산술 연산자 : +,-,/,*,% : 사칙 연산을 위한 것
대입 연산자 : =,+=,-=,/=,*=,%= : 변수에 값을 대입하는 것
조건 연산자 : (식? 식; 식); : 두 개 이상의 값을 비교하여 참,거짓을 판별하고 실행하는 식이 달라지는 것
관계 연산자 : >,<,<=,>=,=,!= : 식의 내용을 크다, 작다, 같다와 같이 비교, 판단할 때 사용하는 것
논리 연산자 : &&, ||, ! : 식의 논리곱(and), 논리합(or), 논리 부정(not)의 연산에 사용하는 것
```
