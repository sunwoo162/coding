#오늘의 코딩
## 1 단어 정리
```
variable |변수| 데이터를 저장하는 공간
constant |상수| 변하지 않는 값
function |함수| 특정 작업을 수행하는 코드 블록
return |반환하다| 함수가 값을 돌려줄 때 사용
parameter |매개변수| 함수가 입력받는 값
argument |인자| 함수에 전달하는 실제 값
statement |문장(구문)| C프로그램의 한 줄 코드
syntax |문법| 코드 작성 규칙
error |오류| 잘못된 코드로 인한 문제
compile |컴파일하다| C코드를 실행 가능한 파일로 변환
debug |디버그하다| 코드의 오류를 찾고 수정
loop |반복문| 코드를 여러 번 실행하는 구조
condition |조건| if 문에서 사용하는 조건
operator |연산자| 계산에 사용되는 기호
integer |정수| 소수점 없는 숫자
float |실수| 소수점 있는 숫자
character |문자| 단일 문자 저장
string |문자열| 여러 문자로 이루어진 데이터
array |배열| 여러 개의 값을 저장하는 자료형
pointer |포인터| 메모리 주소를 저장하는 변수
memory |메모리| 프로그램이 데이터를 저장하는 공간
library |라이브러리| 미리 작성된 코드 모음
header file |해더 파일| C프로그램에서 사용하는 파일
execute |실행하다| 프로그램을 실행하는 과정
print |출력하다| printf()로 화면에 출력
scan |입력받다| scanf()로 사용자 입력 받기
comment |주석| 코드 설명을 위한 텍스트 
identifier |식별자| 변수, 함수, 배열의 이름
data type |데이터 타입| 변수에 저장되는 데이터의 종류
boolean |불리언(참/거짓)|
expression |표현식| 값으로 평가되는 코드
statement |문장| if while return 같은 코드
assignment |할당| 변수에 값을 저장하는 것
modulus |나머지 연산자| % 기호
increment |증가 연산자| ++ 사용 (1증가)
decrement |감소 연산자| -- 사용 (1감소)
bitwise |비트 연산자| & | ^ 사용 (비트 단위 연산)
logical operator |논리 연산자| && || !(AND, OR, NOT)
relational operator |비교 연산자| == != < > 같은 연산자
